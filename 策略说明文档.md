# 小资金多因子精选策略 - 使用说明文档

> 策略文件：`strategy_small_capital_multifactor.py`
> 适用平台：PTrade (恒生量化交易平台)
> 适用市场：中国A股
> 设计资金：20,000 元人民币

---

## 一、策略概述

本策略专为 **2万元小资金** 量身定制，采用"基本面筛选 + 技术面确认 + 严格风控"
的多因子选股框架。核心理念是：**用低频交易降低手续费消耗，用多维过滤提高选股质量，
用刚性止损保护有限本金**。

### 1.1 策略定位

| 项目 | 说明 |
|------|------|
| 策略类型 | 多因子选股 + 趋势跟踪 |
| 交易频率 | **每周一**调仓（低频交易） |
| 持仓数量 | 最多 **2 只**股票 |
| 选股范围 | 沪深300 + 中证500 成分股（约800只） |
| 股价区间 | 3 ~ 25 元（确保资金可操作） |
| 止损线 | -7%（硬止损，每日检查） |
| 止盈线 | +20%（固定止盈） |
| 移动止盈 | 盈利超10%后，从峰值回撤5%触发 |

### 1.2 为什么适合小资金

1. **股价限制3~25元**：2万资金可以买100~600股，仓位分配灵活
2. **每周调仓而非每日**：大幅减少交易次数，降低手续费对收益的侵蚀
3. **最多持2只**：集中持仓，单次交易金额大于5000元，佣金比例更合理
4. **最短持仓3天**：避免频繁进出产生的摩擦成本
5. **选大盘蓝筹+中盘成长**：流动性好、波动适中，适合小资金安全操作

## 二、策略运行流程

### 2.1 整体时序

```
每个交易日
  |
  |-- 09:31  daily_check() 执行
  |     |-- 更新所有持仓的持有天数
  |     |-- 若为周一 → 执行 select_stocks() 多因子选股
  |
  |-- 15:00  handle_data() 执行（日线级别回测）
  |     |-- [每日] check_stop_loss_and_profit()  → 止损/止盈检查
  |     |-- [周一] execute_sell()                → 卖出不合格持仓
  |     |-- [周一] execute_buy()                 → 买入新选标的
  |
  |-- 收盘后  after_trading_end() 执行
        |-- 每周五输出周度汇总报告
```

### 2.2 选股流程详解

```
沪深300 + 中证500 成分股 (约800只)
        │
        ▼
  ┌─────────────────────┐
  │  基本面过滤          │  PE: 5~35, PB: 0.5~6
  │  (估值合理性)        │  排除亏损股、泡沫股
  └─────────────────────┘
        │ 约200~400只
        ▼
  ┌─────────────────────┐
  │  价格过滤            │  股价: 3~25元
  │  (资金可操作性)      │  排除高价股和低价垃圾股
  └─────────────────────┘
        │ 约100~200只
        ▼
  ┌─────────────────────┐
  │  安全过滤            │  排除ST股
  │  (排除高风险标的)    │  排除停牌股（近3日零成交）
  │                      │  排除涨跌停股
  └─────────────────────┘
        │ 约80~150只
        ▼
  ┌─────────────────────┐
  │  四维综合评分        │  估值(30分) + 趋势(30分)
  │  (满分100分)        │  + 动量(20分) + 流动性(20分)
  └─────────────────────┘
        │ 按得分排序
        ▼
  ┌─────────────────────┐
  │  得分 >= 40分       │  取Top 2买入
  │  输出最终买入名单    │
  └─────────────────────┘
```

## 三、四维评分体系详解

### 3.1 估值得分（满分30分）

评估股票的价值是否被低估，便宜的好公司优先。

| 指标 | 满分 | 评分规则 |
|------|------|---------|
| 动态PE | 15分 | PE <= 15：满分15分；PE 15~25：线性递减；PE > 25：快速递减 |
| 市净率PB | 15分 | PB <= 2.0：满分15分；PB 2~4：线性递减；PB > 4：快速递减 |

**设计意图**：优先选择PE低于15倍、PB低于2倍的低估值标的。这类股票通常下行风险
有限，安全边际较高，适合小资金的风险偏好。

### 3.2 趋势得分（满分30分）

确认股票处于上升趋势中，顺势而为。

| 子项 | 满分 | 评分规则 |
|------|------|---------|
| 均线多头排列 | 15分 | 价格 > MA20(+5分)；MA5 > MA10(+5分)；MA10 > MA20(+5分) |
| MACD信号 | 10分 | DIF > DEA即金叉(+5分)；MACD柱放大(+5分) |
| RSI状态 | 5分 | RSI 40~65最佳(+5分)；超买(>75)扣5分；超卖(<25)扣3分 |

**设计意图**：
- 均线多头排列（MA5 > MA10 > MA20）是最经典的趋势确认信号
- MACD金叉确认中期动能向上
- RSI过滤掉超买超卖的极端状态，避免追高或抄底失败

### 3.3 动量得分（满分20分）

选择近期温和上涨的股票，既不追高也不抄底。

| 指标 | 满分 | 最佳区间 | 扣分区间 |
|------|------|---------|---------|
| 近5日涨幅 | 10分 | -2% ~ +5%（温和波动） | > +15% 或 < -10%（暴涨暴跌） |
| 近20日涨幅 | 10分 | 0% ~ +15%（稳健上涨） | > +30%（严重超涨） |

**设计意图**：
- 近期暴涨的股票容易回调，不追；近期暴跌的股票可能有基本面问题，不抄
- 偏好"慢牛"走势的标的，这类股票持续性更强

### 3.4 流动性得分（满分20分）

确保所选股票有足够的成交量，进出顺畅。

| 指标 | 满分 | 评分规则 |
|------|------|---------|
| 日均成交量 | 10分 | 近10日均量 > 50万股：满分；> 20万股：5分 |
| 量比变化 | 10分 | 近5日/近10日量比 1.0~1.5：满分（温和放量）；> 2.0：扣5分（异常放量） |

**设计意图**：
- 小资金虽然不怕流动性问题，但低成交量的股票价格操纵风险大
- 温和放量通常代表资金有序进场，异常放量可能是主力出货

## 四、风控体系详解

风控是本策略的核心。2万本金经不起大幅亏损，**保住本金永远是第一位的**。

### 4.1 三重止盈止损机制

```
                盈利
                 ↑
       +20% ────┤  ← 固定止盈：直接卖出，落袋为安
                │
                │     ← 移动止盈区间：盈利>10%后启动
       +10% ────┤     从最高盈利点回撤5%即触发卖出
                │     例：最高盈利15%，回到10%时卖出
                │
         0% ────┤
                │
        -7% ────┤  ← 硬止损：无条件立即卖出
                │
                ↓
               亏损
```

| 机制 | 触发条件 | 执行时机 | 说明 |
|------|---------|---------|------|
| 硬止损 | 亏损 >= 7% | **每日检查** | 不问原因，坚决执行，保护本金 |
| 固定止盈 | 盈利 >= 20% | **每日检查** | 锁定较大利润，避免利润回吐 |
| 移动止盈 | 盈利超10%后，从峰值回撤5% | **每日检查** | 让利润奔跑的同时锁定大部分收益 |
| 趋势止损 | MA5 < MA20 * 0.98 | **调仓日检查** | 技术面转弱，主动退出 |

### 4.2 买入安全检查

每次买入前均执行以下检查，任一不通过则放弃买入：

1. **涨跌停检查**：涨停板不追（买不到或次日大概率回调），跌停板不接
2. **停牌检查**：近3日成交量为0的不买
3. **ST检查**：股票名称含"ST"的不买
4. **最低得分**：综合评分低于40分的不买
5. **最低金额**：分配资金不足以买1手(100股)的不买

### 4.3 仓位管理

```
总资金 20,000 元
  │
  ├── 持仓1: 最多 ~9,800 元 (约49%)
  ├── 持仓2: 最多 ~9,800 元 (约49%)
  └── 现金缓冲: ~400 元 (2%，预留手续费)
```

- 等额分配：每只股票分配相同金额的资金
- 预留2%现金缓冲，防止因手续费导致资金不足
- 同时最多持有2只，既分散了部分风险，又不会因过度分散导致每笔交易太小

### 5.2 参数调优建议

**保守型（适合新手、极度厌恶风险）：**
```python
g.max_hold_count = 1        # 只持1只，专注管理
g.stop_loss_rate = -0.05    # 5%止损，更严格
g.take_profit_rate = 0.15   # 15%止盈，更快落袋
g.pe_max = 25.0             # 更严格的估值要求
```

```python
g.max_hold_count = 2
g.stop_loss_rate = -0.07
g.take_profit_rate = 0.20
```

**积极型（适合有经验、能承受波动）：**
```python
g.max_hold_count = 2
g.stop_loss_rate = -0.10    # 10%止损，给更多波动空间
g.take_profit_rate = 0.30   # 30%止盈，争取更大利润

```

## 六、PTrade部署指南

### 6.1 回测步骤

1. 登录PTrade平台
2. 进入 **策略** 页面，点击 **新建策略**
3. 将 `strategy_small_capital_multifactor.py` 的全部代码粘贴到编辑器中
4. 配置回测参数：

| 参数 | 推荐设置 |
|------|---------|
| 初始资金 | 20,000 |
| 回测频率 | **日线级别** |
| 开始日期 | 2023-01-01（至少回测2年） |
| 结束日期 | 2025-12-31 |
| 基准指数 | 沪深300 |

5. 点击 **运行回测**
6. 分析回测结果（参考第七节的评估标准）

### 6.2 模拟交易步骤

回测结果满意后：

1. 进入 **交易** 页面，点击 **新建交易**
2. 选择刚才的策略
3. 交易模式选择 **模拟交易**
4. 运行频率选择 **日线级别**
5. 启动模拟交易，**至少观察2周**
6. 关注实际成交价格与回测是否有较大偏差

### 6.3 实盘交易步骤

模拟交易验证通过后：

1. 同上流程，交易模式改为 **实盘交易**
2. 确认账户已入金（至少20,000元）
3. 启动后密切关注前3个交易日的运行情况
4. 确认无异常后可降低关注频率（每周检查一次即可）

### 6.4 注意事项

- **回测与实盘的差异**：回测默认按收盘价成交，实盘可能有滑点
- **佣金差异**：代码中设置的佣金率为0.025%，请确认与您券商的实际费率一致
- **节假日处理**：周一若为节假日，策略会顺延到下一个交易日的周一执行
- **数据延迟**：实盘中 `get_fundamentals()` 的数据可能有1日延迟，策略已做相应处理

## 七、回测结果评估标准

运行回测后，请重点关注以下指标：

### 7.1 核心指标评估

| 指标 | 合格线 | 良好线 | 优秀线 |
|------|--------|--------|--------|
| 年化收益率 | > 5% | > 10% | > 15% |
| 最大回撤 | < 20% | < 15% | < 10% |
| 夏普比率 | > 0.5 | > 1.0 | > 1.5 |
| 胜率 | > 45% | > 50% | > 55% |
| 盈亏比 | > 1.2 | > 1.5 | > 2.0 |

### 7.2 关键检查项

- [ ] 年化收益率是否跑赢沪深300基准指数
- [ ] 最大回撤是否在可承受范围内（建议 < 15%）
- [ ] 交易次数是否合理（太少说明选不到股，太多说明频繁换仓）
- [ ] 单笔最大亏损是否被止损线有效控制
- [ ] 不同市场环境（牛市/熊市/震荡）下的表现是否稳定

### 7.3 回测结果不理想时的优化方向

| 问题 | 可能原因 | 优化方向 |
|------|---------|---------|
| 收益过低 | 选股条件过严/止盈过早 | 放宽PE/PB限制，提高止盈线 |
| 回撤过大 | 止损太宽/持仓太集中 | 收紧止损线到-5%，持仓改为1只 |
| 交易太频繁 | 调仓周期太短/筛选条件波动大 | 改为双周调仓，提高最短持仓天数 |
| 选不到股票 | 价格/估值条件太严格 | 放宽价格区间至3~30元 |
| 胜率太低 | 趋势确认不够/入场时机差 | 提高均线排列要求的权重 |

## 十、手续费影响分析

2万元小资金交易，手续费的影响不可忽视：

### 10.1 A股交易费用构成

| 费用项 | 费率 | 说明 |
|--------|------|------|
| 佣金 | 0.025%（最低5元） | 买卖双向收取 |
| 过户费 | 0.001% | 买卖双向收取 |
| 印花税 | 0.05% | **仅卖出时收取**（2023年8月28日起减半） |

### 10.2 单次交易成本估算

假设买入10,000元的股票：

| 费用项 | 买入 | 卖出 |
|--------|------|------|
| 佣金 | 5.00元（不足5元按5元） | 5.00元 |
| 过户费 | 0.10元 | 0.10元 |
| 印花税 | - | 5.00元 |
| **小计** | **5.10元** | **10.10元** |
| **买卖合计** | **15.20元** | |
| **占交易金额** | **0.152%** | |

### 10.3 年度手续费估算

本策略每周调仓一次，假设每次调仓换1只股票（约5,000~10,000元）：

- 年交易次数：约 52周 x 1次 = 52次（买卖各52次）
- 年均手续费：约 52 x 15 = **780元**
- 占本金比例：780 / 20,000 = **3.9%**

这意味着策略年化收益率需要超过 **4%** 才能覆盖手续费成本。这也是策略选择低频
交易的核心原因。

## 十一、风险提示

### 11.1 策略固有风险

1. **市场系统性风险**：大盘持续下跌时，任何选股策略都难以盈利
2. **因子失效风险**：低PE/PB策略在科技成长行情中可能跑输大盘
3. **小样本风险**：只持2只股票，单只出问题对组合冲击大
4. **回测偏差**：历史回测无法完全代表未来表现（存在过拟合可能）

### 11.2 操作风险

1. **滑点风险**：实盘成交价可能与策略计算价格有差异
2. **流动性风险**：极端行情下可能无法按预期价格成交
3. **系统风险**：PTrade平台故障、网络中断等可能导致策略无法正常执行
4. **数据延迟**：财务数据更新滞后可能影响选股准确性

### 11.3 风险缓解措施

- 每周检查策略运行日志，确认交易执行正常
- 如遇大盘连续下跌（沪深300跌幅超10%），可考虑手动暂停策略
- 定期（每季度）回顾策略表现，必要时调整参数
- 切勿将全部积蓄投入单一策略

## 十二、常见问题 (FAQ)

**Q1: 策略一直选不到股票怎么办？**
> 检查估值参数是否过严。在牛市末期，大部分股票PE/PB偏高，可适当放宽 `g.pe_max`
> 和 `g.pb_max` 的值。

**Q2: 可以改成每天调仓吗？**
> 技术上可以（将 `g.rebalance_weekday` 的判断去掉），但强烈不建议。2万资金每天
> 交易，手续费会严重侵蚀收益。

**Q3: 资金增加后需要修改什么？**
> - 资金到5万：可将 `g.max_hold_count` 改为 3
> - 资金到10万：可改为 3~5，同时放宽 `g.max_price` 至 50
> - 资金到50万以上：建议重新设计策略，引入更多因子和行业分散

**Q4: 为什么不选创业板/科创板的股票？**
> 沪深300+中证500已覆盖A股主要优质标的。创业板/科创板波动更大，且部分有20%
> 涨跌停限制（而非10%），对小资金风险更高。如果想加入，可在 `select_stocks()`
> 中添加相应指数。

**Q5: 止损被频繁触发怎么办？**
> 1. 检查是否处于熊市环境（大盘整体下跌）
> 2. 适当放宽止损线至 -10%（`g.stop_loss_rate = -0.10`）
> 3. 提高趋势确认的权重，减少在弱势中买入

**Q6: 如何查看策略运行日志？**
> PTrade平台的交易页面有日志面板，策略会输出以下标签的日志：
> - `[选股完成]`：每周一选股结果
> - `[买入]`/`[卖出]`：交易操作详情
> - `[止损]`/`[止盈]`/`[移动止盈]`：风控触发记录
> - `[周报]`：每周五的持仓和收益汇总
