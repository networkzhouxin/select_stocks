# 多因子ETF量化策略 V2.5 — 完整说明文档

## 一、策略概述

### 定位
面向小资金（5000元-10万元）的全自动ETF轮动策略。基于7个经典技术指标综合打分，每周从12只跨市场ETF中选出最强的3只持有，通过ATR跟踪止损控制风险。

### 一句话总结
**用7个技术指标给12只ETF打分排名，每周选最强的3只，跌了止损但得分仍高就不卖。**

### 回测业绩

|------|---------|--------|------|---------|------|

**实盘预期：年化8-10%，最大回撤13-16%，可能前1-2年不赚钱。**

## 二、V2.5 新增机制（vs V2.4）

### 1. 止损豁免上限 10%（+9pp）

V2.4 止损豁免无上限：只要得分高就一直不卖。V2.5 加回撤上限——从最高点回撤 <10% 才豁免，≥10% 强制止损，无论得分多高。

```
止损触发 → 得分高 → 回撤 <10% → 豁免
                   → 回撤 ≥10% → 强制止损（force_stopped 防当天买回）
```

### 2. 熊市 A 股仓位减半（+12.8pp）

每天 09:30 用沪深 300 指数 T-1 数据检测熊市：收盘 < MA60 且 MA60 下行 → A 股 ETF 仓位减半。仅影响 5 只 A 股标的（510300/159915/512100/159928/510880），跨市场和跨资产不受影响。10 年触发约 12 次，严格避免误报。

### 3. 去掉国债 ETF

ETF 池从 13 只缩减为 12 只，移除 511010 国债 ETF。国债低波动均值回归特征不适合趋势跟踪框架。

### 4. 其他

- 熊市检测每日执行（轮动日+非轮动日均检测），结果存 `g.market_bearish`
- PTrade 版 `get_deliver` 交割单恢复买入日期（冷启动安全网）
- 盘后日志新增 ATR 和止损价

### 因子列表与权重

| # | 因子 | 权重 | 指标 | 衡量维度 |
|---|------|------|------|---------|
| 1 | 动量 | 0.25 | ROC20（3日平滑） | 涨了多少 |
| 2 | MACD | 0.18 | MACD(12,26,9) | 趋势方向+动能 |

| 4 | RSI | 0.12 | RSI(14) | 相对强弱 |
| 5 | KDJ | 0.12 | KDJ(9,3,3) | 短期超买超卖 |
| 6 | 布林带 | 0.10 | BB(20,2) | 波动率位置 |
| 7 | 成交量 | 0.08 | 3日量/20日量 | 量价配合 |

**权重为设计值，未经优化。** 测试表明策略对权重不敏感（改成0.30/0.05无明显变化）。

### 评分方式：离散分档

每个因子将指标值映射到固定分数档（0-100分），不使用连续线性映射。

**为什么用离散而不用连续？** 离散分档自带"去噪"效果——同档内的ETF得分相同，不会因微小波动触发排名变化。测试表明离散比连续高19pp。

### 评分示例（RSI因子）

| RSI区间 | 得分 | 含义 |
|---------|------|------|
| < 30 | 20 | 弱势 |
| 30-40 | 35 | 偏弱 |
| 40-50 | 50 | 中性 |
| 50-60 | 65 | 偏强 |
| 60-70 | 80 | 强势 |
| 70-80 | 75 | 接近过热 |
| > 80 | 55 | 过热回调 |

### 综合得分
```
总分 = RSI分×0.12 + MACD分×0.18 + 布林分×0.10 + 动量分×0.25
     + 量价分×0.08 + KDJ分×0.12 + 均线分×0.15
```

### 3日平滑
RSI、MACD(DIF/DEA/柱)、KDJ(K/D/J)使用最近3日均值，减少单日噪音。

**所有指标基于T-1日（前一交易日）数据计算，无未来函数。**

## 六、交易决策流程

### 完整流程（每个交易日09:35执行）

```
步骤1：检测止损（仅标记，不执行）
  ↓
步骤2：判断是否轮动日（周二/周四）或有止损触发
  → 否：return（当天结束）
  → 是：继续
  ↓
步骤3：12只ETF全池评分（T-1日数据）
  ↓
步骤4：筛选候选（得分 > 60分）
  ↓
步骤5：决定目标持仓
  - 保护期内持仓（买入<5天）→ 强制保留
  - 得分 > 55分的持仓 → 保留
  - 从候选中补充至max_hold（3只）
  ↓
步骤6：执行止损
  - 触发止损且不在目标中 → 卖出
  - 触发止损但仍在目标中 → 豁免（不卖）
  ↓
步骤7：轮动卖出
  - 不在目标中且非止损触发 → 卖出
  - 停牌 → 跳过
  ↓
步骤8：买入（按得分从高到低排序）
  - 波动率反比调整仓位
  - 100股整手
```

## 七、风控体系

### 6层风控

| 层级 | 机制 | 保护范围 |
|------|------|---------|
| 第1层 | 多市场分散（最多3只不同市场ETF） | 单市场暴跌 |
| 第2层 | ATR跟踪止损（3%-15%） | 单只持仓下跌 |
| 第3层 | 止损豁免（得分高就不卖） | 假止损的手续费损失 |
| 第4层 | 最低持仓期5天 | 买入即卖 |
| 第5层 | 波动率反比仓位 | 高波动品种过度配置 |
| 第6层 | 组合回撤监控 | 实盘异常发现 |

### ATR跟踪止损详解

```
止损价 = 最高价 × (1 - 止损比例)
止损比例 = ATR倍数 × ATR / 最高价
ATR倍数 = 2.0（高波动时） 或 2.5（正常时）
止损比例限制在 [3%, 15%]
```

- **最高价**：用收盘价跟踪，不用盘中最高价（避免上影线噪音）
- **ATR**：每日收盘后用当日K线更新
- **止损每日检查**：不受轮动日限制

### 止损豁免机制

当ATR止损触发时，策略不立即执行，而是先评分确定目标持仓：
- 如果该ETF仍在目标中（得分仍高）→ **不卖**，省两笔手续费
- 如果该ETF已不在目标中（得分已低）→ **执行止损**

**效果：+35pp收益提升。** 约55%的止损触发被豁免（假触发）。

### 仓位计算

```

波动率调整 = min(1.5, max(0.4, 0.15 / 实际年化波动率))

买入股数 = int(实际分配 / 价格 / 100) × 100
```

### 必须做的
- **使用低佣券商**（万0.5以下，最好免5）

### 每次"优化"尝试的结果

| 尝试 | 结果 | 教训 |
|------|------|------|
| 连续化评分 | -19pp | 精确=噪音，粗糙=稳定 |
| ADX自适应权重 | -7pp | 动态权重增加不稳定性 |
| ROC60惩罚 | -46pp | 惩罚中期负动量=错过V型反弹 |
| 空仓机制 | 负 | 清仓后踏空反弹 |
| 趋势持有保护 | 负 | 增加复杂度无收益 |
| QDII量价屏蔽 | 负 | 改变排名无益 |
| 4因子替代7因子 | 负 | 冗余=集成平滑=稳定 |
| ROC3收紧止损 | -62pp | 正常回调被误判为反转 |
| 国债强制兜底 | -18pp | 强制配置低收益资产 |
| 止损后当天不买回 | -10pp | "卖了又买"实际是有效的复核 |
| 止损豁免重置最高价 | -44pp | 下移止损线反而多亏 |

### 成功的改进

| 改进 | 结果 | 为什么有效 |
|------|------|-----------|
| 去掉国债兜底 | +18pp | 不强制买低收益资产 |
| 去掉ADX自适应 | +7pp | 减少复杂度=减少噪音 |
| 止损豁免 | +35pp | 减少无效交易（55%假止损被过滤） |
| **V2.5 止损豁免上限10%** | **+9pp** | 防止得分滞后于价格，极端回撤强制止损 |

## 十四、常见问题

**Q：为什么用7个因子而不是更少？**
A：测试过4因子（只保留独立维度），收益反而下降。7个"冗余"因子提供了集成平滑效果——多个相关因子取均值比单个因子更稳定。

**Q：为什么不用连续评分？**
A：连续评分让排名每天微变，触发更多不必要的换仓。离散分档天然具有"噪音过滤"效果，相当于给排名加了一层额外的换仓门槛。

**Q：为什么周二+周四而不是每天？**
A：每天轮动导致4274次交易亏了91%（V1.0教训）。周二+周四约每2.5天一次，配合最低持仓5天和60分买入门槛，实现了"高频扫描+低频换手"。

**Q：止损豁免不会导致亏更多吗？**
A：豁免只在"得分仍高"时触发——说明其他因子认为该ETF仍然强势，只是ATR止损因短期波动误触发。如果得分真的低了，止损正常执行。

**Q：QDII ETF溢价怎么办？**
A：回测无法模拟溢价，实盘中人工监控。溢价>10%时建议手动暂停该标的的买入。策略代码中不加自动过滤（保持与聚宽回测100%一致）。

**Q：能跑赢沪深300吗？**
A：回测11年累计收益+251% vs 沪深300约+30%。但实盘会打折，预期年化7-9%。长期大概率跑赢沪深300和大部分偏股基金。

**Q：最大能亏多少？**
A：回测最大回撤15.8%，实盘预期18-20%。单只持仓最大亏损15%（stop_cap），3只同时暴跌的理论最大亏损约15%。
